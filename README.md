# L_Shop

Актуальные существующие запросы (для фронта):
Products: 
api/products - получение всех товаров | get
api/products/:id - получение товара по ID | get 
api/prodcts - создание товара | post

Users:
api/users - получение всех пользователей | get
api/users/by-name?name= - получение пользователя по имени | get
api/users/:id - получение пользователя по id | get
api/users - создание нового пользователя | post
